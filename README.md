# Week 1 Lab - Personal Profile Page

This project is a simple personal profile page built with semantic HTML and custom CSS.
I am taking this course to strengthen my web development foundations and get more comfortable building projects from scratch.
My goal is to improve step by step in HTML, CSS, Git, and browser debugging tools.

## Reflection

1. The most surprising thing I learned today was how much semantic HTML improves both structure and readability compared with using generic `div` elements everywhere.
2. One question I still have going into Week 2 is how developers decide when to use flexbox versus grid for page layouts.
